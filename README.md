# Port City Avurudu Game

This is a web-based game application designed and developed for the Port City Avurudu Event. The purpose of this project was to create a fun and interactive experience celebrating the [Sinhala and Tamil New Year](https://en.wikipedia.org/wiki/Sinhalese_New_Year) festival through traditional games.

## Purpose
The game brings the joy of Avurudu celebrations to participants virtually, allowing them to experience traditional festival games in a digital format.

## Features
The application includes two classic Sri Lankan Avurudu games:

1. **Eyeing the Elephant** - A test of precision and focus
2. **Climbing Grease Pole** - A challenge of skill and determination

## Technologies Used
The game was built using the following technologies:

- Game Engine: [Phaser](https://phaser.io/)
- Frontend:
  - HTML5
  - CSS3
  - TypeScript
  - Bootstrap (for responsive UI components)

## Installation & Usage

### Installation
No installation required! Simply host the website files on your preferred web server.

### Usage
1. Open the game in a modern web browser
2. Select your preferred game from the main menu
3. Follow the on-screen instructions for each game

⚠️ **Important Technical Note About Browser Reloads**

🚫 **Do Not Reload the Browser During Gameplay**

**Issue:**  
Reloading during gameplay disrupts touch coordination  
**Effect:**  
Touch inputs won't register accurately in "Aliyata Esa Thabima" (Eyeing the Elephant) until fully reset

✅ **Correct Reset Method:**  
Use the power button in the game's main menu  

The game will:  
- Automatically close  
- Reopen in a new tab  

Touch controls will then work perfectly
